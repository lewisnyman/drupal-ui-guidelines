Buttons

Button styling that can be applied to any element. Use when the user needs to perform an action on the data entered in a form, such as submitting a query, or saving or previewing their content.

.button

<button class="button">Action</button>

.button--primary

Use when a form has a single submission button or more, emphasize the primary button.

<button class="button button--primary">Primary action</button>

.button--small

Sometimes tight spaces recommend the use of smaller controls. Such situations include WYSIWYG toolbars, filter forms, and table rows.

<button class="button button--small">Action</button> <button class="button button--small button--primary">Action</button>

.is-disabled

Use when you provide a button to delete the item you are editing. Only supply one delete button on the screen. Always supply this styling for delete buttons.

<button class="button is-disabled">Button</button> <button class="button button--primary is-disabled">Primary button</button> <button class="button button--danger is-disabled">Danger button</button>

.button--danger

Use when you provide a button to delete the item you are editing. Only supply one delete button on the screen. Always supply this styling for delete buttons.

<button class="button button--danger">Delete</button>
