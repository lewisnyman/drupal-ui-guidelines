Form inputs

A form element is any item within a form used to collect data from a user. Most commonly it consists of a title, an input field, a description, and possibly a prefix, a suffix, or both.

Default Form elements

The title of a form element should be a simple and complete word or phrase which describes the form element. Only the first character of the first word should be capitalized (unless a proper noun or acronym is used). There should be no ending punctuation except for the ending colon.

<div class="form-item">
<label>Text</label>
<input type="text" class="input-text" />
</div>
<div class="form-item">
<label>Textarea</label>
<textarea class="form-textarea"></textarea>
</div>

Disabled form items

<div class="form-disabled">
<label>Title</label>
<input type="text" class="input-text">
</div>

Errored form items

<div class="form-item">
<label class="error form-required">Title</label>
<input type="text" class="input-text error">
</div>

Select elements

Select elements are styled in Webkit only.

<select>
<option value="1">Option 1</option>
<option value="2">Option 2</option>
<option value="3">Option 3</option>
</select>
